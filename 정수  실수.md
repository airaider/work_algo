정수 < 실수

byte < short < int < long < float < double

정수 : 부호비트 + 값비트

실수 : 부호비트 + 지수부비트 + 가수부비트

실수는 연산과정에서 정규화 밑 저장 작업이 많이 들어가므로 overhead가 발생한다
